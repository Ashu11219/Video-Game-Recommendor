# Video-Game-Recommendor
This project involves suggesting the next game you'll like playing based on your recent games and your current mood.

Basic steps-
1. Problem Definition - What am I predicting?

2. Data Collection - Kaggle
3. Data Cleaning - handling missing values, fixing formats, removing duplicates
4. EDA (Explorator Data Analysis) - plot graphs, find patterns, check correlations
5. Feature Engineering - Manipulating data as per necessary
6. Splitting into train, validation and test
7. Model Selection - Linear Regression
8. Training the model - learning paramters
9. Evlation - Validation + Testing
10. Iteration - Improvement
11. Deployment
